# Status Badge Reuse

When showing submission status in the editor, testcase results, or submission UI, reuse `SubmissionStatusBadge` from `modules/submissions/components/submission-status-badge.tsx`.

Status badges should use only shadcn's built-in variants: `default`, `outline`, `secondary`, and `destructive`. Do not use custom `success`, `warning`, or `info` variants for submission statuses.
